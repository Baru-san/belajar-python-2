- untuk melakukan print teks diperlukan tanda petik ("")

    ```python filename="hello_world.py"  showLineNumbers copy
  print("Hello World!")
  ```

## Komentar
komentar adalah barisan yang tidak dibaca oleh komputer yang biasanya digunakan sebagai petunjuk untuk programmer
agar kode lebih mudah dipahami
Komentar dalam Python dimulai dengan #.

#### Contoh:
  ```python filename="hello_world.py" showLineNumbers copy
  # print a number
    print(25)
  ```

### Single-line Comment
  ```python filename="hello_world.py" showLineNumbers copy
        # declare a variable
        name = "John"

        # print name
        print(name)    # John
  ```

### Multiline Comments
  ```python filename="hello_world.py" {1-3} showLineNumbers copy
    # This is an example of a multiline comment
    # created using multiple single-line commenced
    # The code prints the text Hello World
    print("Hello, World!")
  ```

  atau

    ```python filename="hello_world.py" {1-2} showLineNumbers copy
    '''This is an example 
    of multiline comment'''
    print("Hello, World!")
  ```
